What is the difference between Margin, Border, and Padding?

The Margin:>/h2>

"Margin" is a CSS property which sets the margin for all four sides. It is an easy way to set all margins without spending to much time on setting up each side separately. There's margin-top, margin-right, margin-bottom and margin-left - please have a look on the picture.

The Border:

The border property is a shorthand property for:

Border-width

Border-style (required)

Border-color

Please feel free to find the names on the picture.

The Padding:

It is used to create space around the element's content, which can also be defined by the border.
